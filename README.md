configuration.py - содержит конфигурационные параметры для подключения к API:

URL_SERVICE - базовый URL сервера
CREATE_ORDERS - путь для создания заказа
GET_TRACK_NUMBER - путь для получения заказа по его номеру
 
data.py - содержит тестовые данные:
order_body - тело для создания заказа
 

sender_stand_request.py - содержит функции для работы с API:
post_new_orders () - создает новый заказ 
get_track_orders(track) - получает созданный заказ по его трэку


create_track_test.py содержит автотест на проверку успешного получения заказа по его трэку, 
Перед запуском в терминале ввести команду: pip install requests
Тест запускать во вкалдке Testing
